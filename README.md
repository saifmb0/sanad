# Sanad (سند) - Citizen OS

A proactive government services app for UAE citizens and residents. Built for hackathon demo with 100% offline reliability.

## Features

- **Smart Feed**: Proactive alerts for education, visa, traffic, and housing
- **Real Estate Guardian**: Scam detection for property listings
- **Education Support**: Instant enrollment in government programs
- **Digital ID**: Emirates ID card with secure biometric access

## Tech Stack

- Expo (React Native) + TypeScript
- Expo Router (File-based routing)
- NativeWind (Tailwind CSS)
- Lucide Icons

## Getting Started

```bash
npm install
npx expo start
```
